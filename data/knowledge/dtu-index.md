# Index de DTU couramment cités (numéros et domaines)

Les **DTU** (Documents Techniques Unifiés) décrivent les règles de l'art pour l'exécution
des ouvrages. Leur **contenu normatif est édité par le CSTB/AFNOR et payant** : cette fiche
ne donne que l'**index** (numéro → domaine) pour orienter, jamais le texte de la norme.
Toujours se procurer le DTU en vigueur pour l'exécution.

| DTU (repère) | Domaine |
|---|---|
| DTU 13.x | Fondations, dallages |
| DTU 20.1 | Ouvrages en maçonnerie de petits éléments |
| DTU 21 | Exécution des ouvrages en béton |
| DTU 25.41 | Ouvrages en plaques de plâtre |
| DTU 26.1 | Enduits aux mortiers |
| DTU 31.x | Charpente et ossature bois |
| DTU 40.x | Couverture (tuiles, ardoises, etc.) |
| DTU 43.x | Étanchéité des toitures-terrasses |
| DTU 59.1 | Peinture |
| DTU 60.1 | Plomberie sanitaire |
| DTU 65.x | Chauffage / installations thermiques |
| DTU 68.3 | Ventilation mécanique (VMC) |

## Usage

- Citer le bon numéro de DTU dans un CCTP ou un devis technique renforce la crédibilité.
- Ne pas restituer le contenu détaillé du DTU (droits d'auteur) : renvoyer à la référence.
- Vérifier l'indice de version (les DTU sont révisés ; un numéro peut recouvrir plusieurs parties).
